# **App Name**: Navodaya Memories

## Core Features:

- Immersive Landing Experience: Features a captivating hero section titled '7 Years of Memories' to immediately set a nostalgic and emotional tone, replicating the smooth scrolling and section-based layout of the reference.
- Year-wise Journey Sections: Organizes content into distinct, navigable sections for each academic year (Class 6 through Class 12), offering a chronological journey through student life.
- Interactive Student Profile Cards: Displays key student information (photo, name, short bio) within visually appealing, card-based elements, embodying the grid-based and clean UI of the reference.
- Detailed Student Profiles: Upon clicking a profile card, users are taken to a dedicated page featuring comprehensive content including personal memories, favorite quotes, and significant highlights, accessible via smooth transitions.
- Curated Photo Gallery: A dedicated section to showcase a collection of memories through photographs, maintaining the overall minimal and elegant design.
- AI-Powered Memory Prompts: Provides an 'Add Memory' UI (frontend only) which includes an AI tool that generates personalized prompts to inspire users to add their memories effectively.
- Fluid Navigation and Transitions: Ensures a seamless user experience across the application with smooth scrolling between sections and subtle, soft animations on navigation and hover elements.

## Style Guidelines:

- The application will utilize a dark color scheme to align with the premium and nostalgic 'Apple-like' UI of the reference site, ensuring content stands out elegantly.
- Primary color: Muted goldenrod (#DBB261). This hue, rich in yellow-orange warmth, provides a nostalgic yet sophisticated and premium feel, contrasting effectively against the dark background.
- Background color: Deep, subtle warm grey-brown (#1E1D17). This very dark, desaturated background shares the hue of the primary color but ensures a calming and elegant canvas, evoking a sense of enduring memories.
- Accent color: Soft coral (#FFA08F). Analogous to the primary, this vibrant yet gentle shade is employed for interactive elements and highlights, adding a touch of playful warmth and visual interest to the interface.
- Font for all text: 'Inter' (sans-serif). Chosen for its modern, clean, and highly readable qualities, it aligns perfectly with the minimal, premium, and elegant aesthetic, suitable for both headlines and detailed content like memories and quotes.
- Icons will adopt a minimalist, clean line-art style. This choice complements the application's overall elegant and understated design, enhancing usability without clutter.
- A highly structured, grid-based layout, closely mirroring the sectioned and card-based presentation of the reference site. This ensures visual consistency for student profiles and memory galleries, promoting an intuitive user experience.
- Extensive integration of Framer Motion for soft, subtle transitions and fluid animations. This will include smooth scrolling between sections, nuanced hover effects on cards, and gentle page transitions, significantly enhancing the immersive and premium user experience as demonstrated by the reference.
