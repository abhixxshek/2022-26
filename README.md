# Navodaya Memories | Batch 2018-2025

The official digital archive and yearbook for **Jawahar Navodaya Vidyalaya Batch of 2018-2025**. This platform preserves a seven-year legacy of brotherhood, excellence, and shared memories.

## ✨ Features

- **7-Year Journey Timeline**: A cinematic walk-through of every year from Class 6 to Class 12.
- **The Vault (Media Gallery)**: A bulk-uploadable archive for photos with professional image adjustment tools.
- **The Wall (Reflections)**: A public space for final goodbyes and memories, supported by AI-generated prompts.
- **Digital Yearbook**: A searchable directory of all classmates with personalized legacy profiles.
- **Admin Control**: Master moderation tools for archival integrity.

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & Shadcn UI
- **Backend**: Firebase (Firestore, Auth, App Hosting)
- **AI**: Google Genkit (Gemini 2.5 Flash) for memory prompts
- **Animations**: Framer Motion

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+
- A Firebase Project

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the archive.

## 📦 Deployment

This project is optimized for **Firebase App Hosting**. 

1. Connect this repository to Firebase App Hosting in the Firebase Console.
2. The `apphosting.yaml` file is pre-configured for optimal performance and scaling.
